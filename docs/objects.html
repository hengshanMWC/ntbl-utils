<static> tailSpain(o, path, defaultValue)

对象路径查询，避免臃长的 &&
Parameters:
Name Type Description
o array | object 需要查询的对象
path string 路径
defaultValue 当未查询到提供的默认值
Returns:
Type
*
Example
data = {
  a: {
    b: [1, 2, 3, {
      c: 10
    }]
  }
}

tailSpain(data, 'a.b.3.c')  // 10
tailSpain(data, 'a.d')      // undefined
tailSpain(data, 'a.d', 3)   // 3，提供默认值
